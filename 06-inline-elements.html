Ecuaciones de Segundo Grado

Definición

Una ecuación de segundo grado​ o Ecuación Cuadrática de una variable es una ecuación que tiene la expresión general:

ax2 + bx c = 0 , a ≠ 0

donde x es la variable, y a, b y c constantes; a es el coeficiente cuadrático (distinto de 0), b el coeficiente lineal y c es el término independiente.
Este polinomio se puede interpretar mediante la gráfica de una función cuadrática, es decir, por una parábola.
Esta representación gráfica es útil, porque las abscisas de las intersecciones o punto de tangencia de esta gráfica, en el caso de existir, con el eje X son las raíces reales de la ecuación.
Si la parábola no corta el eje X las raíces son números complejos, corresponden a un discriminante negativo y positivo.

Nota:

La notación matemática utiliza caracteres especiales que la mayoría de los editores o procesadores de textos no manejan. Es por ello que se usan extenciones a esos editores o bien, se utiilzan programas específicos para escribir con notación matemática como Mathlab. A la hora de compartir la información se complica porque el otro debe sí o sí disponer de las extensiones o el programa que se utilice.
Es por ello que se ha generalizado el uso de los documentos PDF que son el estandard para compartir información ya que es multiplataforma y con lectores siempre disponibles y gratuitos.
